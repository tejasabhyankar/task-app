 Task App (Full Stack Assignment)

 This is a simple full-stack application where users can:-
 
 Add projects
 
 Delete projects

 Tech Used:-

 Frontend: HTML, JavaScript
 
 Backend: Node.js (Express)

 How to Run:-
 
 Install Node.js
 
 Open project folder
 
 Run:
 
 npm install
 
 node server.js
   
 Open browser:-
 
 http://localhost:3000
  
 Features:-
 
 Add Project
 
 Delete Project
 
 Add Comments
